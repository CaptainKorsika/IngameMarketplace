## Getting Started
To setup you environment, simply use:
```sh
npm install
```
After everything has been installed successfully, execute to setup the backend:
```sh
./gradlew clean build
```
---
## Running Project
To start the Backend Server:
```sh
./gradlew bootRun
```
To start the Frontend Server:
- Open a new terminal and execute:

```sh
npm run dev
```
